== Description ==

Shop Store is a MultiPurpose, Responsive, SEO Friendly, Clean, Customizable and WooCommerce Compatible theme.

== Installation == 

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in 'Shop Store' in the search form and press the 'Enter' key on your keyboard.
3. Click on 'Install' and then 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize the taste.

== Frequently Asked Questions ==

= Where to find theme settings? =
Navigate to Appearance > Customize in your admin panel.
